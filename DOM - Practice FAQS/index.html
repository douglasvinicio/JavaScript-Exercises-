JavaScript FAQs

What is JavaScript?

JavaScript is a browser-based programming language that makes web pages more responsive and sabed round trips to the server.

What is JQuery?

jQuery is a library of the JavaScript functions that you're most likely to need as you develop websites.

Why is jQuery becoming so popular?

Three reasons:

  • It's free.
  • It let's you get more done in less time.
  • All of its functions are cross-browser compatible.
